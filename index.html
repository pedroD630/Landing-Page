Um pouco mais sobre mim

Eu gosto muito de computadores, me perguntei várias vezes como a internet funciona e também sempre tive vontade de fazer meus próprios jogos. Isso me levou a escolher o curso técnico em informática.

Durante 2 anos aprendi HTML, CSS e o básico de JavaScript, Java e mySql. Atualmente, durante o terceiro ano, estou me dedicando a melhorar minha base de JavaScript, estudar um framework (React Js) e aprender a desenvolver aplicativos mobile usando Flutter.

Sempre busquei usar a internet para buscar mais conhecimento. Fiz cursos que me ajudaram muito, como o CS50, de Havard, apesar de não possuir diploma pago, e CS50T (Understanding Tecnology).

Atualmente, meu projeto em destaque é o site Virotic. Das 8 pessoas participantes fui responsável pela programação do site. Utilizamos HTML, CSS, JS e algumas funções do jQuery. O projeto ficou com o segundo lugar de melhor projeto da mostra tecnológica de 2022, na FUNEC Riacho. Códigos no Github.
